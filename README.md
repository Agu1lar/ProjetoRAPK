# RAPK - Relatório Técnico de Manutenção 🛠️

RAPK é um aplicativo desenvolvido em Flutter para facilitar a geração de relatórios técnicos de manutenção diretamente do campo. Com foco em produtividade, o app permite coletar dados, capturar assinaturas digitais e gerar arquivos PDF profissionais de forma instantânea.

## ✨ Funcionalidades

- **📋 Formulário Inteligente:** Coleta de dados do cliente, endereço, equipamento e detalhes do atendimento.
- **✍️ Assinatura Digital:** Captura de assinatura do cliente diretamente na tela do dispositivo.
- **📄 Geração de PDF:** Criação automática de relatórios profissionais em formato PDF.
- **💾 Armazenamento Local:** Consulta de relatórios gerados anteriormente salvos no dispositivo.
- **📊 Indicador de Progresso:** Visualização em tempo real do preenchimento do formulário.
- **📅 Gestão de Datas:** Seleção intuitiva de datas de manutenção.

## 🚀 Tecnologias Utilizadas

- **Flutter & Dart:** Framework para desenvolvimento multi-plataforma.
- **PDF & Printing:** Para criação e visualização de documentos.
- **Signature:** Captura de assinaturas manuscritas.
- **Shared Preferences & Path Provider:** Persistência de dados e gestão de arquivos locais.
- **Intl:** Internacionalização e formatação de datas.

## 🛠️ Como executar o projeto

1. **Pré-requisitos:**
   - Ter o [Flutter SDK](https://docs.flutter.dev/get-started/install) instalado.
   - Um emulador ou dispositivo físico Android/iOS conectado.

2. **Clone o repositório:**
   ```bash
   git clone https://github.com/seu-usuario/rapk.git
   cd rapk
   ```

3. **Instale as dependências:**
   ```bash
   flutter pub get
   ```

4. **Execute o aplicativo:**
   ```bash
   flutter run
   ```

## 📂 Estrutura do Projeto

- `lib/main.dart`: Ponto de entrada e interface principal (Home, Formulário, Visualização).
- `assets/`: Imagens e ícones do projeto.
- `android/`: Configurações específicas da plataforma Android.
